🧑‍💼 Employee Database CLI – Python Project
A command-line employee management system built in Python that supports core CRUD operations using a simple in-memory database structure.

🛠️ Features
Add: Insert new employee records with auto-generated IDs

View: Search employees by ID or full name (case-sensitive)

Update: Modify existing employee information by ID

Delete: Remove employee records and reassign IDs sequentially

Display: List all current employee records in a clean format

Fully interactive CLI with input validation and error handling

🔧 Technologies
Python (no external libraries)

Focused on logic flow, input validation, and user experience

📂 Data Structure
Each employee record includes:

ID (auto-incremented integer)

First and last name

Age

Birth date (stored as a tuple of MM/DD/YYYY)
